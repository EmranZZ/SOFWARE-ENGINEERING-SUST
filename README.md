## 📖 About

This repository serves as a centralized archive for course materials, lecture notes, lab assignments, projects, and exam resources accumulated throughout my BSc Software Engineering journey at SUST. The content spans from the 2nd semester to the 7th semester, covering both theoretical and practical aspects of software engineering education.

**Note:** 🚧 This is an ongoing project. More resources will be added progressively as courses are completed.
